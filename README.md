# Corporate Theme

A premium, professional dark theme for VS Code.

## Features

- **Rich Palette**: Deep navy background with sky blue and emerald accents.
- **Enhanced Readability**: Carefully selected contrast ratios for reduced eye strain.
- **Syntax Highlighting**: Optimized for modern languages like TypeScript, JavaScript, and JSON.

## Installation

### Manual Installation (Recommended for Local Dev)

1. Copy the `corporate-theme` folder to your VS Code extensions directory:
   - **macOS/Linux**: `~/.vscode/extensions/`
   - **Windows**: `%USERPROFILE%\.vscode\extensions\`
2. Restart VS Code.
3. Open the Command Palette (`Cmd+Shift+P` or `Ctrl+Shift+P`).
4. Type `Color Theme` and select `Corporate Dark`.

### Side-loading for Testing

1. Open this directory in a new VS Code window.
2. Press `F5` to open a new [Extension Development Host] window.
3. The theme will be available in the new window.
